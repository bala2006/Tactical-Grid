# Contributing to Tactical-Grid

Thank you for your interest in contributing to Tactical-Grid!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/Tactical-Grid.git`
3. Create a new branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Submit a pull request

## Development Setup

### Prerequisites
- Android NDK
- CMake
- Android Studio or VS Code with C++ extensions

### Building
Follow the instructions in Project.md for setting up the Android build environment.

## Pull Request Process

1. Ensure your code follows the project's coding style
2. Test your changes thoroughly
3. Update documentation if needed
4. Your PR will be reviewed by maintainers
